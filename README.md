# ShopZ — Student Shopping Mobile App UI Design

A modern mobile e-commerce UI concept designed for students, focused on affordable shopping, smooth navigation, and minimal user experience.

---

## Project Overview

**Project type:** Mobile App UI/UX Design  
**Brand name:** ShopZ  
**Target audience:** Students and young online shoppers  
**Tool used:** Figma

## Screens Designed

| Screen | Description |
|--------|-------------|
| Splash Screen | Brand introduction with minimal purple gradient theme |
| Home Screen | Product categories, trending products, and student deals |
| Product Screen | Product details, pricing, ratings, and add-to-cart CTA |
| Wishlist Screen | Saved products with quick add-to-cart option |
| Checkout Screen | Address, coupon section, payment method, and order summary |
| Success Screen | Order confirmation with continue shopping CTA |

---

## Design Style

- Minimal and clean mobile-first layout
- Purple gradient branding for youthful and modern feel
- Card-based product display system
- Rounded UI components for soft visual hierarchy
- Large CTA buttons for accessibility and easy interaction
- Simplified checkout experience for faster purchases

---

## User Persona

**Name:** Riya Thomas  
**Age:** 20  
**Occupation:** College Student  

### Goals
- Find affordable products quickly
- Smooth and fast shopping experience
- Save favourite products easily

### Pain Points
- Complicated checkout flows
- Cluttered shopping apps
- Hard-to-find discounts

---

## Design Decisions

- Purple accent colour creates a modern and energetic identity
- Bottom navigation improves one-hand usability
- Product cards are kept minimal for readability
- Checkout flow is simplified to reduce user drop-off
- Student deals section is highlighted on the home screen
- Wishlist functionality improves repeat engagement
